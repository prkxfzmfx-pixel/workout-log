# 筋トレ記録アプリ

iPhoneのホーム画面から使う筋トレ記録PWA。GitHub Pagesで配信する。

## 使い方（iPhone）

1. SafariでGitHub PagesのURLを開く
2. 共有ボタン → 「ホーム画面に追加」
3. 以降はホーム画面のアイコンから起動（オフラインでも動く）

## 機能

- **記録**: 日付を選んで種目・セット（重量×回数 / 回数のみ / 分）を入力。前回値が自動プリセット
- **体組成**: 体重・体脂肪率を任意入力、除脂肪体重は自動計算
- **カレンダー**: 日ごとの体重と筋トレ●・有酸素●マーク
- **一覧**: 日付ごとのトレーニング内容と体組成
- **集計**: 体重・除脂肪体重の推移グラフ、月別サマリ（ジム日数・総ボリューム・有酸素分）
- **設定**: 種目の追加・リネーム・非表示、JSONバックアップのエクスポート/インポート

## データの保存場所

記録は**端末のブラウザ内（localStorage）**にのみ保存される。サーバーには何も送信されない。
機種変更・別端末との共有は 設定 → エクスポート/インポート で行う。

`data.js` は初回起動時に一度だけ取り込まれる過去データ（きんとれ.xlsx から変換、2025/9/28〜2026/5/13）。
2回目以降の起動では無視される（localStorageが優先）。

## ファイル構成

| ファイル | 内容 |
|---|---|
| index.html | アプリ本体（CSS/JS込みの単一ファイル） |
| data.js | 初回シード用の過去データ |
| sw.js | Service Worker（オフラインキャッシュ、ネットワーク優先） |
| manifest.webmanifest / icons/ | PWAメタデータとアイコン |

## 更新方法

index.html を編集して `git push` するだけ。数十秒でGitHub Pagesに反映される。
（アプリ側はネットワーク優先で取得するので、オンラインで開き直せば新版になる）
